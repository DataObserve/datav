## 在图表上框选 Traces

你可以按着鼠标拖动框选图表上的一段区域，来获取这段区域内的 traces：

<img src="https://github.com/data-observe/assets/blob/main/plugin/select-traces.jpg?raw=true" style={{height: '300px'}}/>

还可以直接点击某个 trace 点 来选择该 trace。


## 对比 Traces 

<img src="https://github.com/data-observe/assets/blob/main/plugin/compare-traces.jpg?raw=true" style={{height: '200px'}}/>

如上所示，你可以在列表中选择两个 trace，然后点击 `Compare` 按钮，来对比这两个 trace 的详情。

<img src="https://github.com/data-observe/assets/blob/main/plugin/compare-traces-modal.jpg?raw=true" />

## Trace 火焰图

火焰图可以帮助你找到某个 trace 中耗时最多的 span，从而找到 trace 中的瓶颈。

<img src="https://github.com/data-observe/assets/blob/main/plugin/trace-flamegraph.jpg?raw=true" />
